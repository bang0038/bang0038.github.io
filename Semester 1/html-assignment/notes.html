Notes

5 Tags That I have Learned

Strong
Defines important text. Example: This text is STRONG!
Superscript
The HTML Superscript element specifies inline text which should be displayed as superscript. Superscripts are typically displayed with a higher baseline using smaller text. Example: Today is October 6th.
Subscript
The HTML Subscript element specifies inline text which should be displayed as subscript. Subscripts are typically displayed with a lowered baseline using smaller text. Example: Water is also known as H2O.
Italic
Makes text italic. Example: Writing in italics makes me feel fancy!
Break
Inserts a single line break. Example: There will be a single line break at the end of this list.
