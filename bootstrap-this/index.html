What is Bootstrap?

The official Bootstrap website describes it as: “The most popular HTML, CSS, and JS framework for developing responsive, mobile-first projects on the web.”

In more simple terms, Bootstrap is a front-end framework that gives coders like you and I access to bits of code, making our lives a whole lot easier. Essentially, Bootstrap means you spend less time bouncing back and forth between html and css and a lot more time designing your website responsively.

Below, you will find examples of all the components that make Bootstrap, Bootstrap!

Let's get started!

5. Grid

So, you've made it this far. You've made it to the grid. The grid component is without a doubt, the most useful and most exciting part of Bootstrap! Do you know how many times that I've built grid systems pre-Bootstrap, only to have my code break somewhere along the way? Do you know how many hours I stayed up trying to perfectly align and fit everything into my container? Never again!

Bootstrap uses a simple and intuitive 12 column grid system that makes designing websites an absolute dream come true!

Here's how it works:

Webpages using Bootstrap are divided into 12 columns by default. When creating a grid system, it's a matter of using simple math to decide how you would like to use the space. Let's say you have 4 cards and you want them to display evenly across the page. Each card would have a column value of 3, because 3x4=12. Now, let's say you wanted 1 card to take up the entire row, in that case it would have a column value of 12! In any scenario, all we are really doing is counting to 12! See? Easy!

The real fun comes in when we make it responsive. You can use values such as col-sm, col-md and col-lg to re-arrange your grid based on the viewport size. Take a look at the example below and re-size your page to see Bootstrap grid responsiveness in action!

Forms. The bane of my existence prior to my discovery of Bootstrap. Who was the professor who had me create forms without Bootstrap again? *cough* Rob? *cough*

In all seriousness, I can't explain the level of gratitude that I have for Bootstrap forms. Creating forms without it was a never ending nightmare. I still have flashbacks today about trying to align the width of the input fields with the send button and playing with borders and padding till midnight... it never worked.
